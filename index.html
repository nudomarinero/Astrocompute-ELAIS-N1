Astrocompute ELAIS-N1

Information about the SKA-AWS Astrocompute proposal called: "Calibration of LOFAR ELAIS-N1 data in the Amazon cloud". Maintained by J. Sabater (Institute for Astronomy, University of Edinburgh).

LOFAR AMIs

We have created an AMI with the LOFAR software pre-installed: ami-173c8c7c. It is based on the Ubuntu 14.04 LTS official AMI.

Software installed:

  • casacore libraries.
  • casarest libraries.
  • pyrap.
  • LOFAR software (release 2.10).
  • CASA.
  • LSMTool.
  • LoSoTo.
  • WSClean.
  • Montage.
  • Some additional Python libraries like astropy, Jupyter, etc.

There is also an AMI (ami-515f3f34) with some more additional software installed that can be used as, for example, a head node. The additional software is:

  • GRID certificates and tools.
  • SRM tools.
  • Postgresql 8.4.10 required by some LOFAR BBS tasks.
